## Get started

<Steps>
  <Step title="Create a Convex project">
    If you don't have one already, run `npm create convex@latest` to scaffold a new Convex project.
  </Step>
  <Step title="Install the component">
    Install `@convex-dev/better-auth` and a pinned version of `better-auth`.

    ```bash
    npm install convex@latest @convex-dev/better-auth
    npm install better-auth@1.5.3 --save-exact
    ```
  </Step>
  <Step title="Choose your framework">
    Follow the guide for your framework to complete installation and configuration.

    <CardGroup cols={2}>
      <Card title="React (Vite)" icon="react" href="/framework-guides/react" />
      <Card title="Next.js" icon="layer-group" href="/framework-guides/nextjs" />
      <Card title="TanStack Start" icon="bolt" href="/framework-guides/tanstack-start" />
      <Card title="SvelteKit" icon="code" href="/framework-guides/sveltekit" />
    </CardGroup>
  </Step>
  <Step title="Add authentication to your UI">
    Use `authClient.signIn.*` and `authClient.signUp.*` methods from Better Auth's client, and wrap your app with `ConvexBetterAuthProvider`.
  </Step>
</Steps>

<Note>
  This component requires **Convex 1.25.0 or later** and **Better Auth 1.5.x**. See the [quickstart](/quickstart) for a complete walkthrough.
</Note>
